# MiSSL

Crie certificados SSL para servidor Apache para testes de scripts em ambiente local no Debian/Ubuntu.

## Documentação

A partir da versão 3 o MiSSL é desenvolvido em PHP e compilado usando o excelente projeto [phpacker](https://github.com/phpacker/phpacker), dessa forma você não precisa ter o PHP instalado em seu computador para executar o MiSSL.

Para usar é bem simples

```bash
chmod +x missl
./missl
```

Digite o número da opção que deseja e siga as orientações que estão no próprio software.
